# homeDX Platform - Folder Structure

## Overview

- `backend/` – NestJS API (PostgreSQL + Prisma)
- `frontend/mobile/hdx_mobile/` – Flutter app (current client)
- `Cube APP V0.1.21/` – Cube Android SDK package/documentation
- `Cube/` – Cube-related reference/license files
- `docs/` – Detailed setup docs
- `deploy.sh`, `stop.sh` – start/stop services
- `setup-wsl-port-forward.ps1` – WSL2 port forwarding (run as Admin on Windows)

## Quick Pointers

- Backend dev server: `backend/`
- Flutter app: `frontend/mobile/hdx_mobile/`
- Cube Android reference package: `Cube APP V0.1.21/`
